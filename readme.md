# interactive_adventure
## Dependencies:
- pip install flask
- pip install nltk
- pip install requests
- pip install numpy
- nltk data, on the Python console (ie. type `python` on the command line): 

    ```
    >>> import nltk
    >>> nltk.download('averaged_perceptron_tagger')
    >>> nltk.download('punkt')
    >>> nltk.download('wordnet')
    >>> nltk.download('brown')
    ```


## Front end:

```
cd front-end
python -m SimpleHTTPServer
```
